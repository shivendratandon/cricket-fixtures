Cricket Fixtures

About:
Python App to get upcoming fxitures from all major cricket playing nations.
Also gives the option to add these to the user's Google Calendar.

Prerequisites:
Python 3 or higher (Usually pre-installed with most operating systems)

How to Use:
1. Run cricket_fixtures.py
2. Enter team name (Not case sensitive)
3. If you want to add fixtures to Google Calendar, go through the authentication process and the events will be added.
