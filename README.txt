Cricket Fixtures

About:
Python Console App to get upcoming fxitures from all major cricket playing nations.
Also gives the option to add these to the user's Google Calendar.

Prerequisites:
Python 3.0 or higher

How to Use:
1. Run cricket_fixtures.py
2. Enter team name (Not case sensitive) Eg. india
3. If you want to add/edit fixtures to Google Calendar, go through the authentication process and the events will be added (The browser window doesn't automatically close safter authentication).
